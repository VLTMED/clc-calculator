# حساب أحمال التبريد والتكييف
## CLC Version 4 A

برنامج احترافي متكامل لحساب أحمال التبريد والتكييف مبني على جداول ASHRAE.

### الميزات
- حساب أحمال التوصيل (جدران، سقف، أرضية)
- حساب الإشعاع الشمسي والزجاج
- الأحمال الداخلية (إضاءة، أشخاص، أجهزة)
- أحمال المنتجات والتخزين البارد
- أحمال الهواء (تسرب، تهوية)
- نتائج فورية بالواط، الكيلوواط، وطن التبريد

### التقنيات
- React 19 + TypeScript
- Vite + Tailwind CSS + shadcn/ui
- RTL عربي كامل

### تشغيل المشروع محلياً
```bash
npm install
npm run dev
```
